# Seminar MMM PFN

Seminar paper on Causal PFNs for Marketing Mix Modeling.

## Research Question

"Can PFNs (CausalPFN, Do-PFN, TabPFN) recover comparable channel attribution and ROAS estimates to a Bayesian MMM without explicitly specified structural priors for adstock and saturation — and where do they systematically fall short?"

## Structure

- raw_data/: original dataset
- processed_data/: cleaned and transformed data
- code/: reproducible Python scripts
- notebooks/: experiments and exploration
- outputs/figures/: generated figures
- outputs/tables/: generated result tables
